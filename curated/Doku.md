## PR workflow
- Run `cpr` to create a PR
- Check labels on PR (minor vs full if applicable)
- Check linter rules in `.sqlfluff`

## Python model setup
- Install Python
- Run `scripts/create yaml file`
- Add `using` clause to the model

## Comparison
